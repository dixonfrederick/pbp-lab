# Lab 2

1. Apakah perbedaan antara JSON dan XML?<br>
- JSON adalah sebuah notasi objek JavaScript, XML adalah sebuah markup language
- Data JSON disimpan dalam map dengan key:value, Data XML disimpan sebagai suatu tree
- JSON digunakan untuk merepresentasikan sebuah objek, XML digunakan untuk representasi data, pengolahan dan pemrosesan dokumen
- JSON mendukung deklarasi array, tetapi tidak mendukung namespace, komentar, dan metadata. Hal sebaliknya berlaku pada XML
- Ukuran file XML lebih besar dan strukturnya rumit, sedangkan file JSON lebih ringan dan sederhana
- XML mendukung tipe data kompleks dan non-primitif, sedangkan JSON hanya mendukung tipe data primitif

2. Apakah perbedaan antara HTML dan XML?<br>
- HTML dikembangkan untuk membuat webpage, sehingga berfokus pada penyajian data. XML dikembangkan untuk pertukaran data lintas platform, sehingga berfokus pada transfer data
- HTML tidak case sensitive, XML case sensitive
- Tag HTML tidak memerlukan penutup, tetapi terbatas pada yang sudah ditentukan. Tag XML memerlukan penutup, tetapi dapat dikembangkan oleh programmer
- Pada HTML, error kecil dapat diabaikan, tetapi tidak demikian pada XML
